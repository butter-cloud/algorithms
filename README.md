## 🧠 Algorithm Study Roadmap

### 1단계: 자료구조 (Data Structures)
-   [ ] 📂 Array / List
-   [ ] 📚 Stack

-   [ ] 🧗 DFS (Depth-First Search)
-   [ ] 🌊 BFS (Breadth-First Search)

-   [ ] 🧵 병합 정렬 / 퀵 정렬

-   [ ] 🔁 재귀 기본 개념

-   [ ] 🧱 Brute-force 기법
-   [ ] 🕹️ 구현 & 시뮬레이션 문제

-   [ ] 🧠 탐욕적 선택 조건
-   [ ] 🪙 대표 문제: 거스름돈, 회의실 배정


### 7단계: 동적 계획법 (Dynamic Programming)

-   [ ] 🧮 대표 문제: 피보나치, 계단 오르기


### 8단계: 그래프 (Graph)
-   [ ] 🧷 인접 행렬 / 인접 리스트
-   [ ] 🧭 위상 정렬
-   [ ] 🚇 최단 경로 (다익스트라, 플로이드 워셜)


### 9단계: 트리 & 힙 & 우선순위큐
